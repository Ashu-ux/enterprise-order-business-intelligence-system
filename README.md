# Enterprise Customer Order and Business Intelligence System

## Overview
A SQL-driven analytics platform to analyze enterprise customer orders,
payments, and revenue trends.

## Tech Stack
- Java
- MySQL
- Advanced SQL

## Key Reports
- Monthly revenue
- Top products
- Customer lifetime value
- Failed transactions
- Delayed payments

## SQL Features
Aggregations, CASE statements, HAVING clauses, indexing, nested queries

## Use Case
Designed to mimic enterprise reporting systems used by solution engineers
to assist business stakeholders.
